# README

This is a boilserplate using rbs & steep on Rails.

## Init

```sh
$ bundle init
$ # add rails to Gemfile
$ bundle install --path vendor/bundle
$ bundle exec rails new . -B -d postgresql --skip-turbolinks --skip-test --api
```

## System Description

* Ruby version: 3.0.1

* Rails version: 6.1.3

* Database: PostgreSQL
